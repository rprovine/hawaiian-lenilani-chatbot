Quick Integration (Copy & Paste)

Add this code to your website, just before the </body> tag:

<!-- LeniLani AI Chatbot -->
<script>
  window.LeniLaniConfig = {
    apiUrl: 'https://chatbot.hawaii.lenilani.com'
  };
</script>
<script src="https://chatbot.hawaii.lenilani.com/widget.js" async></script>
<!-- End LeniLani Chatbot -->

WordPress Integration

For WordPress sites, add to your theme's footer.php or use a plugin:

<?php // Add to footer.php before </body> ?>
<script>
  window.LeniLaniConfig = {
    apiUrl: 'https://chatbot.hawaii.lenilani.com',
    position: 'bottom-right',
    primaryColor: '#F4A261'
  };
</script>
<script src="https://chatbot.hawaii.lenilani.com/widget.js" async></script>

Advanced Configuration

Customize the chatbot appearance and behavior:

<script>
  window.LeniLaniConfig = {
    apiUrl: 'https://chatbot.hawaii.lenilani.com',
    position: 'bottom-left',  // or 'bottom-right'
    primaryColor: '#F4A261',   // Your brand color
    buttonSize: 70,            // Size in pixels
    greeting: 'Aloha! How can we help your business?',
    owner: {
      name: 'Reno Provine',
      email: 'reno@lenilani.com',
      phone: '808-766-1164'
    }
  };
</script>
<script src="https://chatbot.hawaii.lenilani.com/widget.js" async></script>

Testing Your Integration

  1. Add the code to your website
  2. Clear your browser cache
  3. Visit your website
  4. Look for the chat bubble in the bottom corner
  5. Click to open and test the chatbot

Troubleshooting

  • Chat bubble not appearing: Check browser console for errors (F12)
  • CORS errors: Contact support to add your domain to allowed origins
  • Widget loads but won't open: Clear cache and cookies
