How to use this widget on your website:

1. Copy the script tag below and paste it before the closing </body> tag on your website

2. Replace YOUR_RENDER_URL with your actual Render backend URL (e.g., https://your-app-name.onrender.com)

3. The chat bubble will appear in the bottom right corner of your page

Script to copy:

<script>
    // Set your Render backend URL here
    window.LENILANI_CHATBOT_URL = 'YOUR_RENDER_URL';
</script>
<script src="https://your-domain.com/hawaiian-widget.js"></script>

This page is already running the widget - look for the chat bubble in the bottom right corner!
