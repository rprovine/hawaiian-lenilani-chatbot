How to use this widget:

  1. Make sure your backend is running: python run_backend.py
  2. The chatbot bubble will appear in the bottom-right corner
  3. Click it to start chatting with Leni Begonia!

To add this widget to any website:

Copy this code to your website's HTML (update the URL to your deployed backend):

<script>
  window.LENILANI_CHATBOT_URL = 'http://localhost:8000'; // Change to your API URL
</script>
<script src="http://localhost:8000/widget.js"></script>
