WordPress Installation

For WordPress sites, add to your theme's functions.php:

function add_lenilani_chatbot() {
if (!is_admin()) {
wp_enqueue_script('lenilani-chatbot',
'https://chat.lenilani.com/widget.js',
array(), '1.0.0', true);
}
}
add_action('wp_enqueue_scripts', 'add_lenilani_chatbot');

Test the Widget

The chatbot widget should appear in the bottom-right corner of this page. Click the chat button to start a conversation!
