üîç How to Access Your Leads

Option 1: Render Dashboard (Recommended)

  1. Go to Render Dashboard
  2. Click on your hawaiian-lenilani-chatbot service
  3. Click on the "Shell" tab
  4. Run these commands:
# See all lead files
ls logs/leads/

# View all leads in readable format
python -m json.tool logs/leads/*.json

# Count total leads
ls logs/leads/*.json | wc -l

# View most recent lead
ls -t logs/leads/*.json | head -1 | xargs cat | python -m json.tool

Option 2: Download Lead Script

Save this Python script as download_leads.py on your computer:

Option 3: Set Up Email Notifications

Add these to your Render environment variables:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
ADMIN_EMAIL=your-email@gmail.com

Then leads will be emailed to you automatically!
